# Jokes Web App

ASP.NET Core MVC web application developed as an internship project.

## Features

* Create jokes
* View jokes
* Edit jokes
* Delete jokes
* Search jokes
* User authentication and authorization
* Only authenticated users can create, edit, delete, and search jokes
* Hide and reveal joke answers
* User-specific access control

## Technologies Used

* ASP.NET Core MVC
* Entity Framework Core
* SQL Server
* ASP.NET Core Identity
* Razor Views

## Security Features

* Authentication using ASP.NET Core Identity
* Authorization for protected actions
* Restricted access to joke management features

## Project Overview

This application allows users to create and manage jokes in a secure environment. Users can register, log in, create their own jokes, view joke details, search for jokes, and manage existing content. The application restricts sensitive operations to authenticated users.
